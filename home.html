⚙ Settings

Weather

Add a weather card for another city. Set each card's location — Auto (by IP), a city, or precise GPS — with the ⚙ button in its card header. Units apply to all cards.

Clocks

Add a clock for another time zone. Configure each clock — analog/digital, seconds, 24-hour, zone — with the ⚙ button in its card header.

News Feeds

Add any site's RSS/Atom URL. Examples — NYT: https://rss.nytimes.com/services/xml/rss/nyt/HomePage.xml · Hacker News: https://hnrss.org/frontpage · ESPN: https://www.espn.com/espn/rss/news

Task Lists

Add a to-do card — keep several lists (e.g. Work, Home, Groceries). Type a task and press Enter; click the box to complete it, click the text to edit, hover to delete. Each card's options — sort by priority, completed at bottom, hide completed, and nudge after N days (stale-task ⏳ badges) — live under the ⚙ in its header. Set a 🔔 reminder on any task (a toast pops while the dashboard is open). The ✨ button adds AI: brain-dump a paragraph into tasks (it can pull out due times), auto-prioritize & tag, review stale tasks, or plan my day (needs an AI key — add one under the AI Settings tab). Tasks are saved in this browser.
